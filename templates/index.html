    Setup Crunch

    You can use Python setup tools to download and install Crunch automatically. Or you can download the source and use it directly.

    If you have pip installed you can run the following command. 

    $ pip install -e git+https://github.com/dash1291/crunch.git#egg=crunch

    Alternatively, you can download the source by cloning the GitHub respository.

    $ git clone https://github.com/dash1291/crunch.git
